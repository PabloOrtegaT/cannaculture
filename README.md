# Base Ecommerce

Reusable ecommerce foundation built with Next.js, TypeScript, and workspace packages.

## Prerequisites

- Node.js 20+
- npm 10+

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

App URL: `http://localhost:3000`

## Quality checks

```bash
npm run lint
npm run typecheck
npm run test
```
