# Base Ecommerce Docs

This folder contains planning and implementation guidance for the base ecommerce project.

## Structure

- `roadmap/`: Sequential deliverables with scope, tests, and acceptance criteria.
- `research/`: Technology and architecture investigations used to guide implementation.
- `standards/`: Documentation and engineering standards that apply to all deliverables.
- `flows/`: Human-readable documentation for each implemented business flow.
- `erd/`: Domain relationship and schema reference documents.

## How to use

1. Start from `roadmap/00-overview.md`.
2. Read `standards/flow-documentation-standard.md` before implementing.
3. Implement one deliverable at a time.
4. Write flow docs in `flows/` as part of the same deliverable.
5. Do not move to the next deliverable until tests, acceptance criteria, and flow docs are complete.
