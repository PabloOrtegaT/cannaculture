# Flow Docs Index

This folder stores human-readable implementation documentation for important flows.

## Folder convention

- `01-foundation/`
- `02-domain-data/`
- `03-storefront/`
- `04-admin/`
- `05-payments-orders/`
- `06-seo/`
- `07-hardening-release/`

## Rules

1. Follow `docs/standards/flow-documentation-standard.md`.
2. Add or update flow docs in the same deliverable where code changes are introduced.
3. Keep flow docs focused on behavior, rationale, and operations.
